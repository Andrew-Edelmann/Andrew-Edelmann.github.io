1. publishing

This may be interesting if you want to make a website of your own. This one is as barebones as it gets, and it only took a few minutes to set up.

I'm using github's pages feature as a hosting service, which seems to be just about the easiest way to make a static personal website.

I use emacs (in org-mode) to write, and use org-publish to publish org files to html files in the repo mentioned above, and then publishing is simply a matter of running 

git add <filename>
git commit -m "<message>"
git push origin main
# git push -u origin main for first time push in a new repo
